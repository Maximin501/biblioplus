Application web de gestion de bibliothèque développée en PHP.

## 🛠️ Technologies
- **Backend :** PHP 8.x, MySQL
- **Frontend :** HTML5, CSS3, JavaScript
- **Déploiement :** InfinityFree (hébergement), GitHub (code source)

## ✨ Fonctionnalités
- Authentification sécurisée (password_hash)
- CRUD complet des livres
- Gestion des emprunts et retours
- Recherche multicritères
- Tableau de bord statistique
- Interface responsive

## 📂 Installation locale
1. Installer WampServer
2. Cloner le dépôt : `git clone https://github.com/Maximin501/biblioplus.git`
3. Placer dans `C:\wamp64\www\biblioplus\`
4. Créer la base de données avec le script SQL fourni
5. Accéder à `http://localhost/biblioplus/`
